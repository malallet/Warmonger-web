# Warmonger — Constructor de Listas

Web app para crear listas de banda para el juego de miniaturas **Warmonger** de Bad Roll Games.

## Cómo publicarla en la web (sin saber programar)

### Paso 1 — Subir a GitHub

1. Ve a **github.com** e inicia sesión
2. Pulsa el botón verde **"New"** (arriba a la izquierda)
3. Ponle nombre al repositorio: `warmonger-lista`
4. Déjalo en **Public**
5. Pulsa **"Create repository"**
6. Verás una pantalla con instrucciones. Pulsa **"uploading an existing file"**
7. Arrastra o sube todos los archivos de esta carpeta
8. Pulsa **"Commit changes"**

### Paso 2 — Activar GitHub Pages

1. En tu repositorio, ve a **Settings** (la rueda dentada)
2. En el menú izquierdo busca **Pages**
3. En "Source" selecciona **GitHub Actions**
4. ¡Listo! En unos 2 minutos tendrás una URL tipo `https://tuusuario.github.io/warmonger-lista`

### Paso 3 — Cada vez que actualices

Sube los archivos nuevamente a GitHub y en 2 minutos la web se actualiza sola.

## Características

- 14 facciones completas con todas las tropas, armas, hechizos y mejoras
- Validación en tiempo real (límite de puntos, armas de acero, restricciones de facción)
- Funciona desde el móvil
- Exportar/importar listas en texto

## Créditos

- **Juego:** Israel Gutiérrez y Marcos García — Bad Roll Games
- **App:** Herramienta no oficial para facilitar la creación de listas
